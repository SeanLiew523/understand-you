# Feedback Nudge

## Goal
Install lightweight feedback without becoming annoying.

## Use only after meaningful outputs
Do not ask every time.

## Template Options
- “0–10 分就行，这次更接近你想要的状态了吗？”
- “如果只改一处，你最想我改哪里？”
- “这次我最该继续保持的是哪一点？”

## Rules
- ask one short question only
- no guilt
- no repeated nudging after silence
- if user gives score <= 6, classify issue
