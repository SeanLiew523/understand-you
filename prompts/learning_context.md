# Learning Context

## Goal
Understand what the human is learning, how they learn, and how the agent can assist.

## Template
你之前提到学习也是你想让我帮忙的方向。那我想了解一下：

1. 你现在在学什么？或者最近对什么方向特别感兴趣？
2. 你一般怎么学——看文章、看视频、动手做、还是跟人讨论？
3. 你希望我在学习上怎么配合你？比如帮你找资料、整理笔记、当你的讨论搭子、还是别的什么？
