# Work Environment

## Goal
Understand the human's tools, devices, accounts, and technical setup.

## Template
你平时干活用什么设备和工具？我想把这些记下来，这样后面帮你的时候不会搞错环境。

比如：
- 什么操作系统？（Mac / Windows / Linux）
- 常用的核心工具和软件？
- 有没有什么账号需要我区分清楚（哪些是你个人的，哪些是公司的）？
- 有没有什么本地服务或 API 我需要注意的？

你想到什么说什么就行。

## After Reply (CRITICAL — DO NOT STOP HERE)
1. Write to TOOLS.md (environment details).
2. Write to USER.md (tech stack summary).
3. Update coverage: `work_environment = true`.
4. **Bridge to the next gap.**
   Example: "好，环境我记住了。那你手头现在最占心力的具体任务是什么？如果我能帮你扛掉一类事，你最想是哪类？"
5. Continue the conversation — do NOT end here.
